Desenvolvimento de Software por Demanda

Benefícios

  • Uso de tecnologias livres, como Java e (Dojo Toolkit ou JQuery na camada de apresentação web).
  • Roda em qualquer lugar: em web browser, desktop e mobile independentemente do sistema operacional.
  • Interação constante com o cliente.
  • Valores acessíveis para micro empreendimentos.
  • Material de apoio (apostilas e helps).

Características

  • Desenvolvimento muito ágil, usando Scrum e o nosso software WCT.
  • Primeiro protótipo funcional é entrege em uma semana.
  • Sistemas de boa qualidade visual.
  • Independente de plataforma.
  • Independente de sistema gerenciador de banco de dados.

Fábrica de Projetos de Software

Benefícios

  • Os requisitos do sistema são descobertos e documentados.
  • Os riscos e pontos críticos são encontrados.
  • Testes de comportamento da arquitetura do requisitos críticos.
  • Adequação do tempo e prazos para a experiência da equipe.
  • Geração automática de funcionalidades base a partir do modelos do sistema.
  • Estamos em constante aprimoramento com visão à implantação da MPS-Br na empresa.

Características

  • Levantamento da necessidade de qualificação da equipe de produção.
  • Utilização de padrões de mercado como UML e BPMN.
  • Gerência de requisitos.
  • Gerência de configuração.
  • Engenharia de domínio.
  • Rápida prototipação da aplicação.

Geração de Protótipos de Software

Benefícios

  • Visualização do sistema antes de inicar o desenvolvimento.
  • Avaliar a usabilidade enquanto os requisitos são elicitados.
  • Pequenas iterações/sprints.
  • Apresentar os pontos principais do sistema antes de construí-lo.
  • Encontrar interessados em investir no seu negócio.
  • Transformar o protótipo em funcionalidades de uma aplicação.

Características

  • Desenvolvido utilizando tecnologia de ponta.
  • O protótipo é todo construído em modelos utilizando a UML e o MDD.
  • O protótipo é independente de plataforma e de tecnologias.
  • Não há a necessidade de conhecer linguagens de programação.
  • Rica documentação.
