O Work Case Toolkit (WCT) tem como público alvo quem precisa desenvolver um sistema.

Permite aumentar a produção da fábrica de software sem a necessidade de se contratar profissionais altamente capacitados.
Possibilita para a fábrica um rápido retorno de investimento (ROI), de modo que agiliza a produção de sistemas.
Sistema para desenvolvimento de software utilizando modelos. Novidades como MDD e MDA estão inclusas nas soluções disponíveis neste toolkit.
Sistema em conformidade com boas práticas de desenvolvimento de software.
